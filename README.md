# -cmd
**A windows COMMAND LINE client.**

**To get the commands available type 'help' and then press enter.**
#
**Basic commands:**
- clear
- user
- ip ad
- logo
- help

**All commands:**
- cd
- ls
- touch
- rm
- cat
- mkdir
- mv
- cp 
- rn
- nano
- ./
- chmod +x
- sudo

**Sudo commands:**
- touch
- rm
- cat
- apt install
